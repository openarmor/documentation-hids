
.. _OpenArmor-reportd:

OpenArmor-reportd
==============

  ``OpenArmor-reportd`` is a program to create reports from OpenArmor alerts.
  ``OpenArmor-reportd`` accepts alerts on ``stdin``, and outputs a report on ``stderr``.

  .. note::
    Since ``OpenArmor-reportd`` outputs to stderr some utilities like ``less`` will not work if you do not redirect the output.
    End the OpenArmor-reportd with ``2>&1`` to redirect stderr to stdout. ``more`` or ``less`` can be easily used after the stderr redirect.

OpenArmor-reportd argument options
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. program:: OpenArmor-reportd

.. option:: -D <dir>

    chroot to ``<dir>``.

.. option:: -d
      
    Execute OpenArmor-reportd in debug mode. This option can be used multiple times to increase the verbosity of the debug messages.

.. option:: -f <filter> <value>

    Filter the results.

    .. note::

       Allowed filters: group, rule, level, location, user, srcip, and filename.

.. option:: -h

    Display the help message

.. option:: -n <string>

    Create a description for the report.

.. option:: -r <filter> <value>

    Show related entries.

.. option:: -s

    Show the alerts related to the summary.

.. option:: -V

    Display OpenArmor Version and license information.


OpenArmor-reportd example usage
~~~~~~~~~~~~~~~~~~~~~~~~~~~

Example 1: Show Successful Logins
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

    # cat /var/OpenArmor/logs/alerts/alerts.log | /var/OpenArmor/bin/OpenArmor-reportd -f group authentication_success

Example 2: Show Alerts Level 10 and Greater
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

    # cat /var/OpenArmor/logs/alerts/alerts.log | /var/OpenArmor/bin/OpenArmor-reportd -f level 10

Example 3: Show the srcip for all users
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

    # cat /var/OpenArmor/logs/alerts/alerts.log | /var/OpenArmor/bin/OpenArmor-reportd -f group authentication -r user srcip

Example 4: Show Changed files as reported by Syscheck
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

    # cat /var/OpenArmor/logs/alerts/alerts.log | /var/OpenArmor/bin/OpenArmor-reportd -f group syscheck -r location filename
