util.shÂ¶

The util.sh shell script can add a file to be monitored by OpenArmor-logcollector. It can also add a full_command to check for changes to a website, or for changes to the name server of a domain.

A blogpost from Daniel Cid (for 3WoO) introduced this utility.

util.sh argument optionsÂ¶

addfile <filename> [<format>]Â¶

Add a file to be monitored by OpenArmor-logtest. A localfile will be added to the OpenArmor.conf.

addsite <domain>Â¶

Monitor a website for changes. A full_command will be added to the OpenArmor.conf using lynx to dump the initial page. A rule can be written to monitor this output for changes.

Note

Requires lynx.

Warning

This may not be useful on pages with dynamic content.

adddns <domain>Â¶

Monitor the name server of a domain for changes. A full_command will be added to the OpenArmor.conf using host

Note

Requites the host command.
