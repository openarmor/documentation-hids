OpenArmor.conf: Reports options
==========================

Overview 
--------

Supported types 
^^^^^^^^^^^^^^^

Reports options are available in the the following installation types:

* server
* local 

Location 
^^^^^^^^

All reports options must be configured in the /var/OpenArmor/etc/OpenArmor.conf 
and used within the <OpenArmor_config> tag.  

XML excerpt to show location:

.. code-block:: xml 

    <OpenArmor_config> 
        <reports> 
            <!-- 
            Reports options here
            --> 
        </reports> 
    </OpenArmor_config>
